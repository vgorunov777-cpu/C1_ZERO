# Hello World - Внешняя обработка 1С

Простая демонстрационная внешняя обработка для платформы 1С:Предприятие 8.
При нажатии на кнопку **"Нажми меня"** выводится сообщение **"Привет мир"**.

## Структура проекта

```
src/
├── ВнешняяОбработка1.xml                          # Корневой файл метаданных обработки
└── ВнешняяОбработка1/
    └── Forms/
        ├── Форма.xml                               # Метаданные формы
        └── Форма/
            └── Ext/
                ├── Form.xml                        # Описание формы (элементы, команды)
                └── Form/
                    └── Module.bsl                  # Модуль формы (код на BSL)
HelloWorld.epf                                      # Собранный файл обработки
```

## Как использовать

### Вариант 1: Запуск готовой обработки

1. Откройте конфигуратор 1С
2. Файл → Открыть → Выберите `HelloWorld.epf`
3. Запустите обработку (F5)
4. Нажмите кнопку **"Нажми меня"**

### Вариант 2: Сборка из исходников

Для сборки из XML-исходников используется платформа 1С:Предприятие:

```powershell
powershell -File .cursor/skills/epf-build/scripts/epf-build.ps1 `
  -V8Path "C:\Program Files (x86)\1cv8\<версия>\bin\1cv8.exe" `
  -SourceFile "src/ВнешняяОбработка1.xml" `
  -OutputFile "HelloWorld.epf"
```

Если путь к платформе не указан, скрипт автоматически найдёт последнюю установленную версию.

## Результат

При запуске открывается форма с кнопкой **"Нажми меня"**. После нажатия выводится сообщение:

> **Привет мир**
