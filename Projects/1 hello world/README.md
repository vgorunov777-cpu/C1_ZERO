# Hello World - Внешняя обработка 1С

Простая демонстрационная внешняя обработка для платформы 1С:Предприятие 8.

## Структура проекта

- `src/Module.bsl` - Исходный код модуля объекта
- `src/Form.bsl` - Исходный код модуля формы  
- `src/Form.xml` - Описание метаданных формы
- `HelloWorld.epf` - Готовый файл обработки для импорта

## Как использовать

### Вариант 1: Импорт готовой обработки
1. Откройте конфигуратор 1С
2. Файл → Открыть → Выберите `HelloWorld.epf`
3. Запустите обработку (F5)

### Вариант 2: Создание из исходников
1. Создайте новую внешнюю обработку в конфигураторе
2. Скопируйте код из `src/Module.bsl` в модуль объекта
3. Скопируйте код из `src/Form.bsl` в модуль формы
4. Настройте форму по аналогии с `src/Form.xml`
5. Сохраните и запустите

## Результат
При запуске отображается форма с приветствием "Привет, мир! Добро пожаловать в 1С:C1_ZERO"
