Предназначена для настройки адреса РФ.

Открывается из окна настройки адреса в списке Виды контактной информации по команде Дополнительные настройки адреса.

При включенном флажке Только российский адрес становятся активными дополнительные флажки:

  • Запрещать ввод некорректного адреса - если флажок включен, то при некорректном вводе российского адреса невозможно будет ввести данные в программу (имеет смысл только при использовании шаблонов ввода и адресного классификатора). Данные проверяются по ГАР ФНС России.
  • Указывать ОКТМО - включите флажок, для того чтобы автоматически заполнялось поле ОКТМО.
