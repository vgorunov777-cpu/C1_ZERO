# Бенчмарк rlm-tools-bsl

Воспроизводимый бенчмарк на синтетической конфигурации 1С.

## Что измеряется

Время выполнения ключевых операций хелперов:

- Инициализация и построение внутреннего индекса
- Поиск модулей (`find_module`)
- Парсинг BSL (`extract_procedures`, `find_exports`)
- Поиск вызовов (`find_callers`)
- Текстовый поиск (`grep`)
- Парсинг XML метаданных (`parse_metadata_xml`)
- Подписки на события и регламентные задания
- Движения по регистрам

## Как запустить

```bash
uv run python benchmarks/run_benchmark.py
```

## Фикстура

Каталог `fixture/` содержит минимальную синтетическую конфигурацию 1С в формате CF:

- 2 общих модуля (с `#Область`, `Экспорт`, кросс-модульные вызовы)
- 1 документ (реквизиты, ТЧ, `ОбработкаПроведения`, `ОбработкаЗаполнения`, форма)
- 2 справочника (метаданные XML + модули объекта)
- 1 регистр сведений (измерения, ресурсы, модуль набора записей)
- 1 регистр накопления (измерения, ресурсы)
- 1 подписка на событие
- 1 регламентное задание
- 1 перечисление

Всего ~20 файлов. Код синтетический, но использует реалистичные паттерны BSL.

## Ограничения

- Фикстура **не** представляет реальную нагрузку (20K+ файлов ERP)
- Результаты на синтетической фикстуре показывают корректность, не производительность
- Для замеров на реальных конфигурациях используйте `evals/`
